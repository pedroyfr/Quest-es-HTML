Questões HTML

1.Como comentar tags HTML?

Resposta: Todo texto ou mensagem que estiver dentro do "<" e ">" é considerado um comentário.

2.como criar um hiperlink?

Resposta: Para criar um link em uma página da web, você precisa envolver um elemento (um texto, uma imagem e assim por diante) em um elemento de âncora ( ) e definir seu atributo href para que aponte para o URL o qual você quer associar.
